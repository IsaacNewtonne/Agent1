Agent1 Desktop

Local-first personal agent runtime

Quick Start

  1. Start the API server: cargo run -p agent1-cli -- server
  2. In a new terminal: cd desktop && npm run tauri:dev
  3. The desktop app opens automatically and connects to the API.

First Run Checklist

  • ✓ Install Ollama and pull a model (e.g., ollama pull llama3.1:8b)
  • ✓ Verify Ollama: curl http://localhost:11434/api/tags
  • ✓ Create an agent: use the Agent Builder in the desktop UI
  • ✓ Run a task and approve/deny tool calls as needed

Direct API Access

  • API server: http://127.0.0.1:17371
  • Health: GET /api/health
  • Agents: GET /api/agents
  • Sessions: GET /api/sessions

WhatsApp Integration

  • ✓ Connect WhatsApp to receive updates and confirmations
  • ✓ Set AGENT1_PHONE env var to receive notifications
  • cd whatsapp-sidecar && npm start to start sidecar

CLI Reference

  • cargo run -p agent1-cli -- run --agent agents/assistant.toml --task "your task"
  • cargo run -p agent1-cli -- sessions
  • cargo run -p agent1-cli -- memory search "query"
  • cargo run -p agent1-cli -- ui — print these instructions

The legacy static UI has been archived. Use the desktop app or API for all interactions.
